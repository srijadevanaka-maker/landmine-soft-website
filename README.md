Landmine Soft Company Website 🚀

📌 Project Overview

This project is a modern and responsive company website built for Landmine Soft, an IT services company.
The website showcases the company's services, technologies, and career opportunities with a clean and professional UI.

The goal of this project is to design a real-world software company website using modern frontend technologies.

---

🛠️ Technologies Used

- React.js
- HTML5
- CSS3
- JavaScript
- VS Code
- Git & GitHub

---

📄 Pages Implemented

The following pages are included in the website:

1. Home Page
   
   - Company introduction
   - Technologies used
   - Testimonials

2. About Page
   
   - Company vision and mission
   - Team section

3. Services Page
   
   - Web Development
   - Mobile App Development
   - UI/UX Design
   - Backend & API Development

4. Careers Page
   
   - Company culture
   - Job opportunities

5. Contact Page
   
   - Contact form UI
   - Company information

6. Login Page
   
   - Email and password input

7. Register Page
   
   - User registration form

▶️ How to Run the Project

1. Clone the repository

git clone https://github.com/srijadevanaka-maker/landmine-soft-website.git

2. Navigate to the project folder

cd landmine-soft-website

3. Install dependencies

npm install

4. Start the development server

npm start

5. Open in browser

http://localhost:3000
